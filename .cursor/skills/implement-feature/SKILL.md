# Implement Feature

Use this workflow for approved, narrowly scoped feature implementation.

1. Read project docs.
2. Confirm an approved implementation proposal exists.
3. Identify impacted areas.
4. Implement in small steps.
5. Add or update tests.
6. Update relevant docs.
7. Summarize tradeoffs.
8. Open a PR.

If no approved implementation proposal exists, stop and request one before writing code.
