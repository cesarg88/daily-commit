# Product Brief - Daily Commit

## Product

Daily Commit is a personal, portable, score-based daily commitment board.

## Problem

The founder has clear daily goals, but those goals lose visibility, urgency, and weight during the day. Existing task managers, habit trackers, calendars, and dashboards do not behave like a daily commitment board where selected objectives carry score weight and progress is visible throughout the day.

## Audience

The MVP is for one real user first: Cesar, the founder and initial product owner.

## Core Value Proposition

Daily Commit helps the user answer, throughout the day:

- What did I commit to today?
- How is my day going?
- What remains pending?
- What is my current score?
- How consistent was my week?

## Product Principles

- Help the user win the day.
- Treat the day as the main surface of accountability.
- Make commitments visible without relying on memory.
- Keep the Today screen actionable.
- Keep the score honest.
- Prefer clarity over broad configurability.
- Separate weekly performance from weekly consistency.
- Build for one real user before generalizing.

## Non-Goals

- Generic task management.
- Generic habit tracking.
- Social accountability.
- Public templates.
- AI features.
- Automatic health or nutrition integrations.
- Offline-first behavior in the MVP.
